Zero-storage password management

PassForge uses cryptography to derive strong passwords for all your accounts. You only have to remember a single master password. There are no servers that can be offline or hacked, and there are no files to synchronize.

Secure

PassForge uses PBKDF2, a key derivation function that is widely used for encryption. It works by computing a one-way hash function from your master password plus the nickname and repeating the process thousands of times. This makes it resilient even against brute-force attacks where one of your account passwords has been leaked.

You can run PassForge on your own computer or phone. It's open source, and the code is simple enough to verify yourself.

Portable

Because PassForge uses a standardized and widely used algorithm, it is available for most platforms. Implementations are provided for Linux and OS X (in C), for Windows (in Python), for Android (in Java), and for any modern web browser (in Javascript). The C version can run anywhere OpenSSL is available.
